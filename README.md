# Word文档处理脚本

## 功能说明
这个脚本用于处理Word文档，主要功能包括：
1. 将第一行设置为标题（黑体三号加粗）
2. 在标题后添加指定的副标题文本（黑体三号加粗）
3. 处理作者信息（852开头的行，宋体四号）
4. 正文内容设置为宋体小四号
5. 检测并收集文档中的所有图片，将其移动到文档末尾
6. 批量处理文件夹中的所有Word文档
7. 自动重命名处理后的文件，格式为：《作者名》原标题——福州大学先进制造学院与海洋学院关工委2023年"中华魂"（毛泽东伟大精神品格）主题教育征文

## 使用要求
- Python 3.6+
- python-docx 库
